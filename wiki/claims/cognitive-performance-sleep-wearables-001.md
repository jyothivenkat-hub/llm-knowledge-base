# Specific sleep metrics—night-time heart rate, sleep latency, sleep timing, sleep restfulness, and overall sleep quantity—significantly influence cognitive performance, suggesting that sleep quality assessment requires multi-dimensional analysis rather than single metrics.

**Type:** finding
**Source:** Cognitive Performance Sleep Wearables
**Evidence:** Abstract and Introduction state these metrics 'significantly influence cognitive performance' based on 2-month field study data
