# Smartphone keyboard typing metrics can serve as a passive, continuous proxy for cognitive performance measurement, correlating with active alertness tests (PVT) and offering an unobtrusive alternative to active cognitive assessments.

**Type:** method
**Source:** Cognitive Performance Sleep Wearables
**Evidence:** Section 4.3 shows 'changes in alertness (e.g., worse reaction times) correlate with typing metrics (e.g., slower typing pace)' and Introduction emphasizes typing as 'unobtrusive, continuous, and in-situ assessment'
