# High-intensity exercise (SMD = -1.48) substantially outperforms moderate intensity (SMD = -1.06) for sleep improvement, with high intensity achieving SUCRA = 92.9, challenging assumptions that moderate exercise is optimal for sleep.

**Type:** finding
**Source:** Optimal Exercise Dose and Type for Improving Sleep Quality: A Systematic Review and Network Meta-Analysis of RCTs
**Evidence:** Exercise Intensity section: High intensity SMD = -1.48 vs Moderate SMD = -1.06; SUCRA = 92.9
