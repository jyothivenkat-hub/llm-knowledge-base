## Overview

Passive sensing refers to the continuous, unobtrusive collection of behavioral data from users without active participation or intervention. In the context of [[cognitive performance]] monitoring, passive sensing leverages ubiquitous devices—particularly smartphones—to capture behavioral markers that reflect underlying physiological and cognitive states. This approach contrasts with active testing methods by enabling real-world, longitudinal data collection that integrates seamlessly into daily workflows.

## Key Findings

### Typing Metrics as Cognitive Proxies

Smartphone keyboard typing characteristics demonstrate a robust correlation with active cognitive performance measures. Specifically, typing metrics (including typing pace and keystroke patterns) correlate with the Psychomotor Vigilance Test (PVT), a gold-standard measure of alertness. This relationship establishes typing behavior as a valid passive proxy for cognitive performance, offering practical advantages in continuous monitoring scenarios where active testing is impractical.

### Sleep-Alertness-Typing Transitive Relationship

A critical finding is the parallel influence of [[sleep quality]] on both alertness and typing metrics. Sleep quality establishes a transitive relationship where sleep influences alertness, and sleep independently influences typing characteristics through similar mechanisms. This parallel pattern suggests that typing metrics can serve as an indirect proxy for sleep quality, enabling multi-level inference about physiological state from a single behavioral signal.

### Continuous Data Collection Without User Burden

The key methodological advantage of passive typing sensing is the elimination of user intervention. Unlike active cognitive assessments that require explicit user participation and interrupt workflow, passive keystroke collection occurs continuously and transparently. This enables ubiquitous computing applications where behavioral data complements or potentially replaces traditional cognitive testing protocols.

## Methods

Passive sensing of cognitive performance has been operationalized through:

- **Keystroke dynamics analysis**: Continuous monitoring of typing pace, keystroke intervals, and pressure patterns on smartphone keyboards
- **Behavioral data aggregation**: Collection of typing metrics across naturalistic smartphone use without explicit user prompts
- **Correlation validation**: Comparison of passively-collected typing metrics against active cognitive tests (PVT) to establish validity
- **Multi-signal integration**: Linking typing behavior to sleep quality measures to establish causal and correlational pathways

## Open Questions

- **Generalizability across populations**: Do typing-cognition correlations hold across different age groups, typing proficiencies, and device types?
- **Confounding factors**: How do factors like typing task type, emotional state, or physical fatigue independently influence keystroke patterns?
- **Temporal dynamics**: What is the optimal sampling frequency and aggregation window for detecting meaningful changes in cognitive state?
- **Clinical applicability**: Can passive typing sensing detect clinically significant cognitive decline or sleep disorders with sufficient sensitivity and specificity?
- **Privacy-utility tradeoff**: How can continuous keystroke monitoring be implemented while maintaining user privacy and consent?
